# ferroclaw

> A personal AI assistant built in Rust — inspired by OpenClaw.

ferroclaw（**ferro**=铁/Rust + **claw**）是 OpenClaw 的 Rust 复刻版，目标：单一二进制，低内存（<20MB），快速启动（<100ms）。

## MVP 能力

聊天 · 规划 · 记忆（短/长） · Linux 命令执行
